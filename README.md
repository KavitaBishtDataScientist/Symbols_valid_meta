# Symbols_valid_meta
Exploratory Data Analysis (EDA) on the Symbols Valid Meta dataset to understand stock market securities, exchanges, ETFs, financial status, market categories, and trading patterns using Python, Pandas, Matplotlib, and Seaborn.
## Symbols Valid Meta - Exploratory Data Analysis (EDA)
### Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Symbols Valid Meta dataset. The dataset contains metadata about stock market securities listed across different exchanges.

The main goal of this project is to understand:

Different types of listed securities
Stock exchange distribution
Financial status of companies
ETF vs non-ETF securities
Market categories and trading information
Missing values and data quality issues
Overall structure and patterns in the dataset

This project helps in understanding how stock market metadata is organized and how companies are distributed across exchanges and categories.

### Objectives
Understand the structure of the dataset

Handle missing values
Analyze stock exchanges and market categories
Study ETF and non-ETF distributions

### Key Insights
Most securities are listed under major exchanges like Nasdaq and NYSE.
Financially stable companies dominate the dataset.
Distressed companies represent only a very small portion of the data.
ETFs form an important category among listed securities.
Market categories help classify companies based on size and listing standards.
Missing values in financial status were highly concentrated and required preprocessing.
